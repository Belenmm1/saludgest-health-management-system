# SaludGest

Sistema de gestión hospitalaria orientado a mejorar la experiencia del paciente y optimizar el flujo clínico.

## Funcionalidades
- Terminal de autogestión
- Sala de espera con clasificación
- Mapa de camas
- Hoja de ruta del paciente

## Tecnologías
- HTML
- CSS
- JavaScript

## Objetivo
Brindar una solución accesible, clara y eficiente para instituciones de salud.

## Estado
Posibilidad de escalabilidad.
Sujeto a mejoras
