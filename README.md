#  Medigest - Sistema de Gestión Hospitalaria

Este proyecto es una plataforma integral de gestión administrativa y clínica diseñada para optimizar los flujos de trabajo en centros de salud. Como **Analista de Sistemas**, desarrollé esta solución enfocada en la escalabilidad (modelo SaaS) y en la mejora de la experiencia del usuario (UX) para el personal médico.

### Funcionalidades Destacadas:
- **Tablero de Control (Dashboard):** Visualización de métricas críticas en tiempo real (pacientes en espera, ocupación de camas, alertas de auditoría).
- 
- **Mapa de Camas e Internación:** Gestión visual y dinámica de la disponibilidad por sectores (UTI, Guardia, Piso).
- 
- **Hoja de Ruta del Paciente:** Sistema de seguimiento paso a paso desde la admisión hasta el alta.
- 
- **Terminal de Autogestión:** Interfaz simplificada para que el paciente gestione su llegada de forma autónoma.
- 
- **Panel de Auditoría Médica:** Control de protocolos y cumplimiento de tiempos de atención.

### Tecnologías y Arquitectura:
- **Frontend:** HTML5, CSS3 profesional (uso de variables, CSS Grid y Flexbox para diseño responsivo).
- 
- **Lógica de Negocio:** JavaScript (Vanilla) para la navegación entre módulos, gestión de estados de la terminal y sistema de notificaciones.
- 
- **Enfoque:** Arquitectura limpia diseñada para ser integrada con backends robustos (Python/Java) y bases de datos SQL.
